  • 组合问题:用一元兑换1分、2分、5分的硬币，兑换数量为50枚，问可以有多少种组合的方式?(嵌套for循环 +计分板变量)
    var count = 0;
    for (var one = 0; one <= 50; one++) {
        for (var two = 0; two <= 50; two++) {
            for (var five = 0; five <= 20; five++) {
                if (one + 2 * two + 5 * five == 100 && one + two + five == 50) {
                    document.write("one=" + one, "two=" + two, "five=" + five, "<br>");
                    count++;
                }
            }
        }
    }
    document.write(count);
