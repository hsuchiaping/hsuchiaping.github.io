JS第一天上课

2022-04-13

JS概述

软件架构模式

  1. C/S （Client Server）客户端/服务器
    • 用户需要下载客户端才能使用
    • 例子：QQ、微信、LOL
  2. B/S （Browser Server）浏览器/服务器
    • 使用浏览器就可以使用功能，所有程序放在服务器上
    • 例子：京东、淘宝、哔哩哔哩

URL

网址

  • URL(Uniform Resource Locator) 统一资源定位符，用于定位互联网上的资源，确定要访问资源的位置
  • 格式：协议://主机:编号/资源路径，比如https://www.jd.com/index.html
    1. 协议：http或https
    2. 主机：
      IP：4段数字组成，每段数字取值为0~255，例如192.168.3.125，定位网络上的计算机
      域名：简化IP难以记忆的问题
      端口号：一般隐藏状态，默认为80http或者433https;

历史

为了解决浏览器中对用户数据进行校验的问题以及实现动态网页效果的技术而产生。由网景公司开发，标准为：ECMAScript。

概述

  1. 作用
    1. 实现用户数据校验与交换
    2. 实现页面的动态效果
  2. 组成
    1. ECMAScript 最核心、最基础的部分，所有浏览器都能执行。
    2. BOM(Browser Objctive Model) 浏览器对象模型，提供与浏览器交换的接口。
    3. DOM(Document Objctive Model) 文档对象模型，提供与网页文档交换的接口。
  3. 版本
    1. ES5（主流）所有主流浏览器都支持，IE9支持部分，IE11 IE12完美支持。
    2. ES6（辅） 2015年后的浏览器支持，IE都不支持，EDGE支持。
  4. 运行环境
    1. 浏览器 所有浏览器脚本解释引擎默认为JavaScript。
    2. Node.Js 独立运行JS代码的服务器环境。

如何在HTML中使用JavaScript

  1. 内部
    在<head></head>或者<body></body>标签中使用，<script>JS代码</script>
    <head> <script> JS代码 </script> </head> <body> <script> JS代码 </script> </body>
  2. 外部文件
    通过在<head></head>或者<body></body>标签中使用<script src="JS文件路径"></script>引入写好的JS文件。

注意

JS代码从上到下，从左到右执行，严格区分大小写、中英文，建议每条JS命令后加;分号作为结束标志。作为引入外部标签的<script>的里面再写JS代码不会执行。

注释

进行注释说明的文字，不会被执行，提高代码可读性
单行注释：Ctrl + /；注释效果：//单行注释
块注释：Shift Alt + A 注释效果:

     /*块
        注
        释*/

文档注释：/**注释内容 */，一般加在函数、类定义前。

输入与输出

  1. 输入
    1. prompt('提示内容')
    2. 获取到的内容为字符串类型。
  2. 输出
    1. 页面输出 document.write(输出内容)
    2. 控制台输出 console.log()
    3. 弹框输出 alert()

加号+的作用

  1. 当+链接的操作数有任一个为字符串类型时，该符号起到拼接的作用。
  2. 当操作数都为数字时，则是数学运算+的作用。

变量

概念：向系统申请一个储存空间，并命名，之后可以通过使用名字来使用储存空间

  1. 语法
    1. 不赋值 var num1,num2
    2. 赋值 var num = 1, num2 = 2或者var num1 = prompt('请输入一个数字'), num2 = prompt('请输入第二个数字')
  2. 取值
    1. 将变量里的值写出来
    document.write(`两数相加结果为：${num1 + num2}, 两数相乘结果为：${num1 * num2}, 两数相除的结果为：${num1 / num2}`);

标识符

所有命名都要遵循标识符规则：

  1. 只能由数字、字母、下划线_、美元符号$组成，不能使用其他的符号、中文等等，如demo*
  2. 不能以数字开头，如2demo
  3. 不能与关键字冲突，如var

驼峰命名法

  1. 大驼峰，有多个单词组成的命名，每个单词的首字母大写，其他小写
    1. 语法 DaTuoFeng
    2. 类名使用大驼峰
  2. 小驼峰，多个单词的命名，第一个单词全小写，后面的所有单词首字母大写
    1. 语法 xiaoTuoFeng
    2. 变量名、函数名使用小驼峰
  3. 作用：见名知意

数据转换

number与string之间的转换

  1. number转string
    1. 使用+做字符连接，给数字用"+"连接一个字符串，就可以实现转换
var num = 1;
var str = num + "";
typeof str;
//string

2. `目标数据.toString()`

var num = 1;
var str = num.toString();
typeof str;
//string

3. `目标数据.tiFixed(小数位数)`，主要针对有小数的数字，有小数位数的字符串

var num = 1;
var str = num.toFixed(2);
console.log(str)
//1.00
typeof str;
//string
  1. string 转 number
    1. 字符串转为小数或者整数，按照数据的字面值进行转换，如果数据不是数字类型，则输出结果为NaN
      Number(目标字符串)
var str = '123.123';
var num = Number(str);
console.log(num);
//123.123
typeof num;
//number

2. 字符串转整数

parseInt(目标字符串)
如果，字符串不是数字或者正负号开头，则输出为NaN
反之，以数字或者正负号开头的，从左到右，直到第一个非数字的字符或者直到字符串末尾，将匹配的部分转为整数。舍去小数部分。主要用于有小数的字符串转为整数。

var str = "123a";
var num = parseInt(str);
console.log(num);
//123
typeof num;
//number
var str = "a123";
var num = parseInt(str);
console.log(num);
//NaN
typeof num;
//number

3. 字符串转含有小数，浮点型

parseFloat(目标字符串);
如果目标字符串不是以数字、正负号、小数点开头的，输出结果为NaN；
反之，以数字、正负号、小数点开头的，从左到右，直到第一个非数字、第二个小数点、字符串末尾，将其前面部分转为小数

//情况1：数字，两个小数点
var str = '123.123.123';
var num = prseFloat(str);
console.log(num);
//123.123
typeof num;
//number

//情况2：数字 小数点 非数字
var str = '123.12asdasdasd';
var num = prseFloat(str);
console.log(num);
//123.12
typeof num;
//number

//情况3：负号，小数点，数字 字母
var str = '-.12.66asdasdasd';
var num = prseFloat(str);
console.log(num);
//-0.12
typeof num;
//number

//情况4：小数点 数字 字母
var str = '.12asdasdasd';
var num = prseFloat(str);
console.log(num);
//0.12
typeof num;
//number
