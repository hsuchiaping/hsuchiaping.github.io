# JS 第三天

2022-04-17

switch

一般用于固定的判断。比如月、周类似的这种有固定条件数
语法：

switch(变量){
    case 值 :
        执行条件
    case 值 :
        执行语句;
        break
    ...
    default :
        break
}

while

当条件满足开始循环，满足就会一直循环，直到不满足时停止。

while(condition){
    执行的语句
}

do while

执行一次再判断条件是否满足。满足则继续循环并且判断条件。

do{
    执行的语句
}while(condition)

for

满足条件，就执行。不满足就退出。
i= 1 ：餐宿初始化
i< x : 循环条件
i ++ : 步进

for(i = 1; i< x ; i++>){
    执行的语句
}

break 和 continue

break：结束本层循环
continue： 结束本层本次循环，进行下次循环

使用总结

如果需要至少循环一次，使用dowhile
如果是死循环，则用while
其他大部分情况用for
