animation动画

添加动画： 
@keyframes 动画名{
    from {
        动作
    }
    to{
        动作
    }
}
或者
@keyframes 动画名{
    0%{
        动作
    }
    .
    .
    .
    100%{
        动画
    }
}

动画过渡效果设置：
复合属性：

animation: name duration delay timing-function iterating-count direction fill-mode

动画播放状态:animation-paly-state

练习：轮播图，设置展示窗口，容器加载图片，动画呈现换页的效果。

响应式布局

适应不同设备屏幕。
语法：

<link type="stylesheet" href="" media="设备类型名（screen、TV等等）">
style{
   @media 媒体类型 and/ , /only/not（描述媒体的特点）{
        选择器{
            样式
        }
    } 
}
