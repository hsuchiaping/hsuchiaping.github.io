数组API

2022-04-21

API：Application Programming Interface

应用程序接口
var arr = [1,2,3,4,5,2]

  1. unshift 在数组开头添加一个元素，并返回新数组的长度
console.log(arr.unshift(1));    //5
console.log(arr);   //[1,1,2,3,4,5]
  1. shift 删除数组开头的元素，并返回被删除的元素
console.log(arr.shift());   //[1]
console.log(arr);   //[2,3,4,5]
  1. push 在数组末尾添加元素，并返回新数组的长度
console.log(arr.push);   //[6]
console.log(arr);   //[1,2,3,4,5,6]
  1. pop 删除数组末尾的元素，并返回被删除的元素
console.log(arr.pop());     //[5]
console.log(arr);       //[1,2,3,4]
  1. splice 添加或者删除数组中的指定的元素，并返回被删除的元素数组；
    参数：
    1. 开始的位置
    2. 删除的个数
    3. 添加的元素
//删除
console.log(arr.splice(1));     //[2]
console.log(arr);     //[1,3,4,5]
//添加
console.log(arr.splice(1,2,3,3))    //[2,3]
console.log(arr)    //[1,3,3,4,5]
  1. join将数组转为一个字符串，并返回该字符串
    参数：字符串元素间连接的符号
console.log(arr.join("-"));     //"1-2-3-4-5"
  1. slice截取数组的部分，组成新的数组并返回
    参数：
    1. 开始的位置（要去）
    2. 结束的位置（不取）
console.log(arr.slice(3,5));       //[4,5]
console.log(arr);       //[1,2,3]
  1. concat 合并两个数组到新数组，并返回
    参数：
    1. 要合并的数组
    2. 要合并的数组
var newArr = [];
console.log(concat(arr));      //[1,2,3,4,5]
  1. reverse 反转数组的顺序
console.log(arr.reverse());     //[5,4,3,2,1]
  1. indexOf 查找某个值在数组中第一次出现的下标并返回，不存在返回-1
console.log(arr.indexOf(2));       //1
console.log(arr.indexOf(0));       //-1
  1. lastIndexOf 查找某个值在数组中最后一次出现的下标并返回，不存在返回-1
console.log(arr.lastIndexOf(2));       //5
console.log(arr.lastIndexOf(0));       //-1
  1. 回调函数
    在声明函数时，在函数内调用函数，在参数内加上相对应的形式参数，在调用函数时，函数实际参数改为具体函数名。或者在调用函数时，实际参数声明函数。
function myFunction(arr,method){
    for(i=0;i<arr.length;i++){
        method(arr[i],i,arr)
    }
}
function myFunction1(val){

}
myFunction(arr,myFunction1)
myFunction(arr,function(val){

})
  1. forEach遍历数组内的每个元素，并将其放到回调函数使用
    1. 参数：回调函数
    2. 回调函数的参数：
      1. 要操作的元素
      2. 元素的索引
      3. 数组
    3. 返回值：无
arr.forEach(function(val){
    执行的代码
})
  1. filter 过滤，遍历数组中的每个元素，执行回调函数，将回调函数为true的当前元素添加到新数组，循环完毕，返回新数组。
    1. 参数：回调函数
    2. 回调函数参数：
      1. 要操作的元素
      2. 元素的索引
      3. 数组
    3. 回调函数返回值：boolean
    4. 返回值：过滤元素组成的新数组
arr.filter(function(val){
    return 执行代码
})
  1. map对数组所有元素执行回调函数，将回调函数的返回值储存到新数组，循环完毕，将新数组返回。
  2. 参数：回调函数
  3. 回调函数参数：
    1. 要操作的元素
    2. 元素的索引
    3. 数组
  4. 回调函数返回值：任意
  5. 返回值：回调函数返回值组成的新数组
arr.map(function(val){
    return 执行代码
})
  1. sort对数组进行排序，执行回调函数，将回调函数的返回值作为排序方式。
  2. 参数：回调函数
  3. 回调函数参数：
    1. 要操作的元素1
    2. 要操作的元素2
  4. 回调函数返回值：>0 or <0;
    1. 大于0，则将a放在后面
    2. 小于0，则将a放在前面
  5. 返回值：排好序的数组
arr.sort(function(a,b){
    return a-b;
})
  1. some检查数组中是否有任意一个元素通过测试。对每个元素执行回调函数，有任意一个回调函数的返回值为true，则循环结束，返回结果为true。
  2. 参数：回调函数
  3. 回调函数参数：
    1. 要操作的元素
    2. 元素的索引
    3. 数组
  4. 回调函数返回值：boolean;
  5. 返回值：boolean；根据回调函数返回值确定。
  6. every检查数组中是否所有元素通过测试。对每个元素执行回调函数，有任意一个回调函数的返回值为false，则循环结束，返回结果为false。
  7. 参数：回调函数
  8. 回调函数参数：
    1. 要操作的元素
    2. 元素的索引
    3. 数组
  9. 回调函数返回值：boolean;
  10. 返回值：boolean；根据回调函数返回值确定。
  11. reduce将数组元素数量减为1个，对每个元素执行回调函数，将每次回调函数的返回值作为下一次回调函数的参数传入，循环结束，最后一次回调函数的返回值作为reduce的返回值。
  12. 参数：回调函数，初始参数（param）
  13. 回调函数参数：
    1. param
    2. 要操作的元素
    3. 元素的索引
    4. 数组
  14. 回调函数返回值：旧的param;
  15. 返回值：最后一次回调函数的返回值。
arr.reduce(function(param,val){
    return param = param + val
},传入回调函数的初始参数)
  1. includes 查数组中是否包含指定元素。
  2. 参数：
    1. 要找的元素
    2. 开始索引，默认为0
  3. 返回值
    1. boolean，存在为true，不存在为false
arr.includes(1,2)
