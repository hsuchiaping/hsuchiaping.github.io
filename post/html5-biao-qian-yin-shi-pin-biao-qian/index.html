HTML5新特性——音视频标签

2022-03-29

音视频标签

音频标签

使用<audio>给网页添加音频。

代码及相关属性：
<audio src="url" controls loop muted autopaly></audio>

属性：

  1. controls 添加控件
  2. loop 设置循环播放，默认只播放一次
  3. muted 设置默认静音
  4. autoplay 设置自动播放

视频标签

使用<video>给网页添加视频。

代码及相关属性：
<video src="url" controls loop autoplay murted>

属性：

  1. controls 添加控件
  2. loop 设置循环播放，默认只播放一次
  3. muted 设置默认静音
  4. autoplay 设置自动播放
