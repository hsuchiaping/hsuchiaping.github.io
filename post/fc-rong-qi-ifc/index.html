FC容器——IFC

2022-03-29 FC容器

IFC

概念：

Inline Formatting Context 行内格式化上下文 （行盒），是一行渲染区域规则，规定一行内元素如何排列。

特点：

  1. 在IFC中，盒子水平放置的，排列一个接一个，从包含块的边界开始；
  2. 在IFC中，盒子水平方向的margin、padding是有效的；
  3. 这些盒子有多种对齐方式，底部、基线、中线等等；
  4. IFC负责的矩形区域称为行盒，整个一行的显示区域。
  5. 行盒宽度由一行内所有的内容决定；
  6. 高度由元素最高点到最低点决定。

影响IFC的css属性

  1. font-size
  2. font-family
  3. height
  4. line-height
  5. vertica-align：设置行级元素在垂直方向上的对齐方式
