数组

概念

一组有序数据的集合，每个数据有对应的编号：下标、索引。可以利用变化快速检索数据。

  1. 索引
    1. 有数字组成
    2. 取值范围：[0,数组长度)
  2. 元素
    1. 数组中存储的数据

语法

  1. 声明数组
    1. 空数组
      1. var arr = [];
      2. var arr = new Array();
    2. 非空数组
      1. var arr = [1,2,3,4];
      2. var arr = new Array(1,2,34,4);当括号内只有一个数字时，会被识别为数组长度
  2. 向数组中添加数据
    arr[0] = 123;
  3. 取出数组中的数据
    console.log(arr[1]);
  4. 获取数组长度
    console.log(arr.length);
  5. 删除最后一个元素
    arr.length--;
  6. 将指定元素赋为空empty
    delete arr[1];
  7. 数组遍历
    1. for(i = 0;i <arr.length;i++){}
    2. for(var index in arr)
    3. for(var n of arr)
